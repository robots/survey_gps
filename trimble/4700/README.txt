4700 receiver:
130may23-alt.47s - fixed week rollover on 1.30 official firmware.


To load this firmware you need Winflash program from trimble, serial cable and free com port on your system.

After installing Winflash you need to import trim-130.reg file into Windows registry - double click, and confirm.
Copy the 130may23-alt.47s file into winflash install directory( C:\program files\trimble\winflash\firmware). This
will register the firmware with winflash. 
Afterwards you can select the firmware to flash from Winflash.
